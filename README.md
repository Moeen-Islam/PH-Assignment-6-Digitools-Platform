# 🚀 DigiTools – Premium Digital Tools Platform

## 📌 Description
DigiTools is a modern web application that showcases and sells premium digital products. Users can browse tools, add them to a cart, and complete a smooth checkout experience with a clean and responsive UI.

---

## ⚙️ Technologies Used
- ⚛️ React.js  
- 🎨 Tailwind CSS  
- 🧩 DaisyUI  
- 🔔 React Toastify  
- 🌐 JavaScript (ES6+)  

---

## ✨ Features
- 🛒 Add to cart with real-time updates  
- 📱 Fully responsive design (mobile, tablet, desktop)  
- 🎯 Interactive UI with smooth animations and notifications
